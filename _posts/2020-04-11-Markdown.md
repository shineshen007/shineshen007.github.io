## 缘由
经常写博客需要用到markdown，记录一些语法，方便查看。

## 标题
```
# 这是一级标题
## 这是二级标题
### 这是三级标题
#### 这是四级标题
以此类推,#符号后面要空一格
```
## 换行
在句尾空格  
两个字节，另起一行书写

## 字体
```
**这是加粗的文字**
*这是倾斜的文字*`
***这是斜体加粗的文字***
~~这是加删除线的文字~~
```

## 图片
```
![图片alt](图片地址（网页地址比较稳定） ''图片title'')

图片alt就是显示在图片下面的文字，相当于对图片内容的解释。
图片title是图片的标题，当鼠标移到图片上时显示的内容。title可加可不加
```

## 超链接
```
[超链接名](超链接地址 "超链接title")
title可加可不加
```
