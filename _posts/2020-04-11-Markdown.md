# 标题

在行首使用#表示标题，标题级别由#的数量决定，最多支持六级标题。例如：
```
# 一级标题
## 二级标题
### 三级标题
```

# 文字格式

在书写文字的时候，我们需要对文字的格式进行一些修改，或者需要对重点内容进行标注以表示强调。对文字格式的变换包括：加粗、斜体、更改颜色、更改字体、高亮、上下标、增加下划线等等。

上述的功能在Markdown语法中都是支持的。

## 加粗/斜体

```
*斜体文字*
**粗体文字**

斜体和加粗可以同时使用
***加粗与斜体***
```
*斜体文字*

**粗体文字**

***加粗与斜体***

## 字体种类/颜色/大小

### 字体类型
参数face表示设置字体类型
示例：
```
<font face=楷体>楷体字</font>
<font face=黑体>黑体字</font>
```
<font face=楷体>楷体字</font> 


<font face=黑体>黑体字</font>


### 字体颜色
参数color表示设置字体颜色, 可以使用颜色英文名或者十六进制颜色值
示例：
```
<font color=blue>蓝色</font>
<font color=#008000>绿色</font>
<font color=red>红色</font>
```
<font color=blue>蓝色</font> 


<font color=#008000>绿色</font>


<font color=red>红色</font>

### 字体大小
参数size表示设置字体大小, 取值范围是1-7, 数字越大字体越大
示例：
```
<font size=5>5号字</font>
<font size=8>8号字</font>
```
<font size=5>5号字</font> 


<font size=8>8号字</font>

### 字体类型\颜色\大小可以同时设置

font指令中同时设置face、color和size参数
示例：
```
<font face=楷体 color=green size=4>楷体绿色4号字</font> 
```
<font face=楷体 color=green size=4>楷体绿色4号字</font>

# 列表

Markdown支持有序列表和无序列表
## 无序列表

使用星号*、加号+或是减号-作为列表标记，这些标记后面要添加一个空格，然后再写内容。
例如：
```
* 第一项
* 第二项
* 第三项

(*, +, -显示的效果相同)
```
* 第一项
* 第二项
* 第三项


## 有序列表

有序列表使用数字加上. 再打一个空格即可。
例如：
```
1. 第一项
2. 第二项
3. 第三项
```
1. 第一项
2. 第二项
3. 第三项


## 列表嵌套

嵌套列表只需要在子列表前面打上两个或四个空格即可（可使用Tab键）。
子列表若是有序列表，则会自动调整成其他字母表示顺序。
例如：
```
1. 第一项
    1. 第一项的第一个元素
    2. 第一项的第二个元素
```
1. 第一项
    1. 第一项的第一个元素
    2. 第一项的第二个元素

子列表若是无序列表，则会自动调整成其他符号表示。
例如：
```
- 第二项
    - 第二项的第一个元素
    - 第二项的第二个元素
```
- 第二项
    - 第二项的第一个元素
    - 第二项的第二个元素

      
## 表格

基本语法
使用|分隔单元格，使用-创建表头分隔线。例如：

| 表头1 | 表头2 |
|-------|-------|
| 单元格1 | 单元格2 |
| 单元格3 | 单元格4 |


文字对齐方式
-:表示内容与标题右对齐
:-表示内容与标题左对齐
:-:表示内容与标题居中
例如：

| 表头1 | 表头2 |
|:-------|:-------|
| 单元格1 | 单元格2 |
| 单元格3 | 单元格4 |
