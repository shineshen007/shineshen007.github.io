# 简介

这是一个 R Markdown 文档，它支持普通 Markdown 和嵌入 R 代码。

## 绘制图表

以下是一个示例散点图：

```{r echo=TRUE, warning=FALSE}
# 生成数据
x <- 1:10
y <- rnorm(10)

# 绘图
plot(x, y, main = "示例散点图", col = "blue", pch = 16)
```
