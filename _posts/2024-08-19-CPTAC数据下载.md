# 下载PDC-client
[下载链接](https://pdc.cancer.gov/pdc/data-download-documentation)，这里以Linux端为例，上传压缩文件到服务器文件夹内，unzip解压缩

# 下载数据
设置工作路径，上传Manifest文件到路径下，在终端输入
```
sudo ./pdc-client download -m PDC_file_manifest.tsv
```
